# 🛒 E-commerce Web App

A simple e-commerce web application built with **HTML, CSS, JavaScript**, and **Firebase** for backend services (authentication, database, and storage).

## 📁 Project Structure


## 🚀 Features

- 🔐 User authentication (Register & Login)
- 🛍️ Browse products
- 🛒 Add to cart & view cart
- ❤️ Wishlist system
- 📦 Admin panel to:
  - Manage products
  - View and manage orders
  - Manage categories

## 🧰 Technologies Used

- **HTML5**
- **CSS3**
- **Vanilla JavaScript**
- **Firebase** (Auth, Firestore DB, Hosting, Storage)
